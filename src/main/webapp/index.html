概述

一路走来，我们走过很多弯路，有过喜悦，有过失落，但是不变的是坚持。在不断地探索、发现和总结中，我们持续完善自己，完善团队的管理规范。

  • Project Manage System

    PMS(高分项目管理系统)

    PMS是一个项目过程管理一体化平台。
    提供需求管理、软件设计、开发过程管理以及缺陷追踪等完整的项目生命周期管理支持。

    系统漫游

    PMS主要功能包含基于角色的权限控制、项目活动、新闻、讨论区、版本库、文档管理以及便捷的文件分享。 PMS是面向用户的设计，系统会将您所关注的信息汇集到用户工作台，并且会通过EMAIL向您发送最新的变更情报。您所需要做的就是Check邮件或者直接登陆到您的工作台处理和分配项目任务。

  • Code Review System

    CR(高分代码评审系统)

    CR是一个基于GIT的代码评审平台。
    提供代码评审、CI集成、分支管理等完整的代码质量管理支持。

    系统漫游

    CR主要功能包括基于Git的代码评审机制、基于角色的权限控制、代码差异查看以及分支管理合并。 CR的设计基于Git分布式版本控制系统，当开发者向版本库提交代码，系统会自动将其提交到评审平台（代替默认直接提交到版本库中的操作），经过既定的审核流程后代码会被合并到版本库中，并且会在CI上执行一次自动构建。

  • Continuous Integration System

    CI(高分持续集成系统)

    CI是一个基于Huson的持续集成平台。
    提供自动构建、自动部署、自动化测试等完整的软件发布管理支持。

    系统漫游

    CI主要功能包括基于角色的权限控制、可定制的构建部署以及自动化测试。 CI是面向程序的设计，在用户体验上不足，但是有着强大的定制扩展支持，因此由SCM直接管理，不对普通用户开发配置。管理员可以根据需要定制构建的周期、条件以及方式，适应软件的自动发布，提高部署发布效率。

  • Code Review System

    VPS(高分虚拟主机服务)

    VPS是一个基于虚拟化技术的虚拟主机服务平台。
    提供虚拟主机的租凭、虚拟实验环境的搭建等完善的虚拟化管理方案。

    系统漫游

    VPS主要功能包括虚拟主机管理、虚拟主机迁移、快照服务以及3389远程控制服务。 当项目生命周期中需要使用服务器搭建实验环境就通过简单的一些授权使用VPS，她能够快速的虚拟化生成所需的实验硬件环境，并且后期管理变更十分方便。

  • Email Manage System

    Gmail(高分邮件管理系统)

    Gmail是一个邮件管理系统，是Grid Mail的简称。
    提供smtp、imap、pop3以及web mail等完整的邮件服务支持。

    系统漫游

    Gmail主要功能包括支持传统的smtp/pop3服务、imap即时邮件服务以及在线的web mail服务。 Gmail是面向用户的设计，有良好的用户体验，优美的UI界面，您可以通过web mail或者熟悉的邮件客户端方便的管理您的日常邮件。

  • Instant Messaginge System

    IM(高分即时通讯系统)

    IM是一个网络即时通讯系统。
    提供消息通讯、联系人管理、群组管理以及邮件提醒等完整的即时消息通讯服务支持。

    系统漫游

    IM主要功能包括在线消息通讯、离线消息留言、文件传输、语音会议、联系人/群组管理以及基于邮件的通知提醒服务。 您可以通过统一身份认证ID登陆IM，与实验室内所有成员沟通、协作，享受无限的即时通讯服务。

  • File Transfer Protocol System

    GFTP(高分文件传输系统)

    GFTP是一个文件传输服务平台。
    提供文件上传、下载、目录索引等完整的文件传输服务。

    系统漫游

    GFTP主要功能包括基于FTP的文件传输服务、文件分享、QoS以及目录检索。 GFTP是您日常办公以及项目开发过程中不可或缺的一部分，您可以在GFTP主站下载操作系统镜像、软件、文档等整理的各类资源；同时，您也可以通过上传与大家分享您所拥有的有价值的资源。

  • Subversion System

    SVN(高分版本管理系统)

    SVN是一个基于C/S结构的集中式版本管理系统。
    提供变更管理、里程碑管理、分支管理等完整的版本管理控制支持。

    系统漫游

    SVN主要功能包括基于用户名/密码的安全控制、便捷易用的变更管理。 基于集中式设计的SVN，非常容易上手，简单的提交、更新、编辑冲突即可完成日常的工作，实现简单的版本管理控制，适合刚接触版本控制的同学使用。

  • GIT System

    Git(高分分布式版本管理系统)

    Git是一个分布式版本控制管理系统。
    提供基于SSH的认证、变更管理、里程碑管理、分支管理等完整的版本管理控制支持。

    系统漫游

    Git主要功能包括基于SSH的安全控制、灵活的分支/里程碑管理。 基于分布式设计的Git，在学习路线上较为陡峭，复杂而又繁多的命令，但是其优秀的设计思想，使Git灵活而又富有扩展性。在您接受她之后，就会体会到一种秀丽的异域风采。

  • Wikipedia System

    wiki(高分知识库系统)

    wiki是一个知识管理系统。
    提供公众协同、知识分类、知识版本管理等完整的知识库管理支持。

    系统漫游

    wiki主要功能包括基于角色的权限管理、便捷的知识分类管理、以及基于公众协同模型的知识版本管理。 wiki是实验室最宝贵的资产，汇集无限的解决方案、百科知识、技术学习教程，有了她您在日常的研究工作和学习肯定会事半功倍。

  • Demo System

    系统漫游

    demo主要功能包括软件功能示例、算法程序示例以及前端界面展示。 demo是一个软件虚拟实验室，囊括了非常多常用的程序demo以及实验室成员在学习过程中积累分享的实验程序；demo同样也是在日常研究和学习过程中必不可少的一部分。

  • Application Programming Interface

    API(高分API开放平台)

    API是一个应用程序接口文档综合平台。
    提供各类开源程序在线API文档以及实验室成员发布的软件API文档。

    系统漫游

    API开放平台主要功能包括提供API文档的在线浏览、检索和打包下载。 API开放平台是在开发过程中一个得力的助手，能帮助您快速的找到相关的接口描述和程序示例；除此之外，您还可以通过打包下载功能得到离线版的API文档，让您能够随时、随地查阅API。
